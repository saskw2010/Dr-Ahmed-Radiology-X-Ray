# Dr Ahmed Radiology — X-Ray

مستودع مستقل للدكتور أحمد النجار، مخصص لوضع «بروف» ومهام الأشعة التشخيصية.

## الهدف

حفظ التعليمات التشغيلية، قوالب التقارير، فهرس الحالات السابقة، ومراجع الأشعة في مكان واحد، مع فصلها عن هوية مصطفى ومشروعات Sky365.

## نقطة البداية

- مفتاح التشغيل: `بروف` أو `\\X-ray`
- اللغة الافتراضية: English
- المخرج الافتراضي: Word `.docx` قابل للتعديل والطباعة
- الأسلوب: Structured consultant-style radiology report
- الأقسام الأساسية: Patient Data, Technique, Findings, Impression
- لا يتم اختلاق أي finding غير موجود في البيانات أو الصور.
- لا يُرسل التقرير النهائي تلقائيًا؛ يُرسل عند طلب المستخدم صراحةً.

## المحتويات

| المسار | الغرض |
|---|---|
| `instructions/` | التعليمات والقواعد التشغيلية |
| `templates/` | قوالب التقارير حسب modality |
| `reports/` | سجل التقارير والحالات السابقة |
| `research/` | مراجع وبروتوكولات البحث |
| `archive/` | ملفات Word/PDF/DICOM التي ستُستورد لاحقًا |
| `metadata/` | فهارس المصدر وحالة الاكتمال |

## حالة المستودع

هذه هي النسخة الأولى المبنية من الذاكرة والسياق المتاحين. الملفات الأصلية القديمة ليست موجودة في مساحة العمل الحالية، لذلك لم يتم اختلاق نسخ بديلة لها. أسماء الملفات المعروفة ومسارات الاستيراد موثقة في `metadata/source-inventory.md`.

## ضابط الجودة

قبل اعتماد أي تقرير سريري نهائي: مراجعة الصور الأصلية/DICOM والبيانات الإكلينيكية بواسطة الطبيب المختص.
